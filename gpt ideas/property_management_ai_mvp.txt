PROPERTY MANAGEMENT AI MVP – FULL SPEC

========================
1. MVP GOAL
========================
Tenant issue → AI categorization → contractor outreach → quote collection → landlord decision.

Do NOT build:
- payments
- contractor marketplace
- AI calling
- accounting integrations

========================
2. CORE USERS
========================
- Property Manager
- Tenant
- Contractor

========================
3. MVP FEATURES
========================

A. Tenant Intake
- Web form submission
- Fields: property, unit, description, photo, contact info

B. AI Classification
Outputs:
- category
- subcategory
- urgency
- summary
- confidence

Categories:
- Plumbing, Electrical, HVAC, Appliance, Structural, Pest, Lock/Security, Other

Urgency:
- Emergency, Urgent, Routine, Low

C. Contractor Dispatch
- Send to preferred vendors
- SMS/email with structured request

D. Contractor Response
- Simple form:
  - price
  - availability
  - notes
  - inspection required

E. Dashboard
- Issue summary
- urgency
- contractors
- quotes
- recommendation
- approve button

F. Recommendation Logic
Based on:
- price
- availability
- vendor priority
- response quality

========================
4. 2-WEEK BUILD PLAN
========================

Week 1:
- Auth, properties, contractors
- Issue form
- AI classification
- Dashboard base
- Contractor dispatch
- Response form

Week 2:
- Approval flow
- Notifications
- Status system
- Edge cases
- UI polish
