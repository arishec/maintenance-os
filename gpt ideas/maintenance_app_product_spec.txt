PRODUCT SPEC – MAINTENANCE TRIAGE & QUOTE COORDINATION

Version: MVP v1.0
Purpose: Developer handoff document

==================================================
1. OVERVIEW
==================================================
This product enables property managers to efficiently handle maintenance issues by:
1. Collecting tenant issues (text + photo)
2. Using AI to classify category and urgency
3. Sending structured requests to contractors
4. Aggregating contractor quotes
5. Recommending a vendor
6. Allowing one-click approval

Primary goal:
Reduce time from issue submission → vendor selection.

Platform:
Responsive web application (NOT native mobile app)

==================================================
2. USERS
==================================================
1. Property Manager (primary user)
2. Tenant (issue submission only)
3. Contractor (quote response only)

==================================================
3. CORE WORKFLOWS
==================================================

A. Tenant Submission
- Submit issue via web form
- Upload image
- Provide description + contact info
- System stores issue
- AI classifies issue

B. Contractor Outreach
- Manager reviews issue
- Selects contractors
- System sends request (SMS/email)
- Contractors respond via link

C. Vendor Approval
- Manager compares quotes
- System shows recommendation
- Manager approves vendor
- Notifications sent
