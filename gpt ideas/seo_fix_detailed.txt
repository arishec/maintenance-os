ISSUE #1 — WRONG DOMAIN (MOST IMPORTANT)

You are using:

https://ifbids.com

in:
- app/layout.tsx
- app/robots.ts
- app/sitemap.ts
- canonical URLs in pages

If this is NOT your real domain:

THIS IS A MAJOR SEO PROBLEM.

Why:
- Google indexes wrong domain
- canonical signals point elsewhere
- sitemap URLs are wrong
- Open Graph URLs are wrong

--------------------------------------------------

FIX (DO THIS EXACTLY)

Step 1:
Set environment variable:

NEXT_PUBLIC_SITE_URL=https://your-real-domain.com

--------------------------------------------------

Step 2:
In app/layout.tsx:

Replace:
const siteUrl = process.env.NEXT_PUBLIC_SITE_URL || 'https://ifbids.com';

With:
const siteUrl = process.env.NEXT_PUBLIC_SITE_URL!;

(Do NOT fallback to wrong domain)

--------------------------------------------------

Step 3:
In app/robots.ts and app/sitemap.ts:

Replace all hardcoded domains with:

const siteUrl = process.env.NEXT_PUBLIC_SITE_URL!;

--------------------------------------------------

Step 4:
REMOVE hardcoded canonicals like:

canonical: 'https://ifbids.com/pricing'

Replace with:

alternates: {
  canonical: '/pricing'
}

Let metadataBase handle full URL.
